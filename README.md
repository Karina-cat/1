# Movie Library

**Автор:** Карина Гафарова
**Вариант:** "Movie Library"
**Дата:** май 2026

## Описание

Простое приложение для хранения личной кинотеки. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Реализована фильтрация по жанру и году. Данные сохраняются в формате JSON.

## Технологии

- Python 3.x.
- Tkinter (стандартная библиотека).
- JSON.
- Git.

## Использование

1. Клонируйте репозиторий.
2. Запустите `main.py`.
3. Добавляйте фильмы через форму.
4. Используйте фильтры для поиска.
5. Данные сохраняются автоматически.

## Примеры использования

- Добавить фильм: *«Начало», «Фантастика», 2010, 8.8*.
- Отфильтровать по жанру «Комедия».
- Загрузить данные из `movies.json`.
